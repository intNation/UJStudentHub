# Overview

**UJ Student Hub** is a centralized platform for University of Johannesburg students to discover and engage with:

- Student societies and clubs  
- Day residences  
- University events  

The system simplifies student engagement by allowing students to join communities, RSVP for events, and receive notifications, while giving admins tools to manage memberships, events, and reports.

---

# Features

**For Students:**  
- Discover and join communities and events  
- RSVP to events and track attendance  
- Receive event notifications  

**For Admins:**  
- Manage communities and memberships  
- Submit and track event proposals  
- Generate event and community reports
